## 오늘의 생각

- 코드리뷰를 해준건 이번이 처음이다. 어색하고 상대방에게 어떤말을 해야 잘 전달할 수 있을지 고민이 컸다. 그러나, 상대방의 코드를 보는 건 정말 큰 도움이 되었다. 지금 이대로 페어프로그래밍이나 코드리뷰받는식으로 하면 좋겠으나 코로나 행정명령이 떨어져서 어떻게 될지 모르겠다.
- S3수업과 병행하여 Github Action을 이용한 CI/CD를 진행중이다. CI는 성공적으로 끝냈으나 CD에서 계속해서 문제가 생긴다. Github Action은 진행이 자연스럽게 되어 깔끔하게 S3로 전달이 되었다. 그러나, S3에서 받아서 CodeDeploy 하려고 하는데 에러가 여러개가 난다. 문제는 에러가 다른사람이 봐도 처음보는 에러라고 한다. 6시간정도 시간을 부었지만 결국 성공못한게 슬슬 열받는다.
- CodeDeploy 설정문제인지 appspec.yml 문제인지 감이 안잡힌다. 지금은 해당 디렉토리를 지우고 실행하면 작동하지만 그대로 다시 커밋 푸시하면 NotEmptyError가 또다시 발생할 듯 하다. CodeDeploy는 정상 작동하는걸 보니 CodeDeploy의 IAM 설정 문제는 아닌거 같고, appspec.yml을 살펴보자.
